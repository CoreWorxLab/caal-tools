# espn-epl

Get Premier League scores, standings, and team schedules from ESPN.

## Voice Triggers

- "Hey Cal, What are the Premier League scores?"
- "Hey Cal, Where is Arsenal in the table?"

## Required Services



## Setup

No environment variables required.

No credentials required.

## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "espn-epl"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s espn-epl
```

## Usage

Premier League data suite - scores, standings, and schedule. Parameters: action (required) - 'scores', 'standings', or 'schedule'; team (optional for scores/standings, required for schedule) - team name like 'Arsenal', 'Liverpool', 'Spurs'. Examples: 'Premier League scores', 'where is Arsenal in the table', 'when do Liverpool play next'.
