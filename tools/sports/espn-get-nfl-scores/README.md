# espn-get-nfl-scores

Gets live NFL game scores with optional team filtering, including game status and current score information.

## Voice Triggers

- "Hey Cal, What are the NFL scores?"
- "Hey Cal, How are the Packers doing?"

## Required Services



## Setup

No environment variables required.

No credentials required.

## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "espn-get-nfl-scores"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s espn-get-nfl-scores
```

## Usage

Get live NFL scores. Optional parameter: team (e.g., 'Packers', 'Bears') to filter results. Examples: 'NFL scores', 'how are the Packers doing', 'did the Chiefs win'.
