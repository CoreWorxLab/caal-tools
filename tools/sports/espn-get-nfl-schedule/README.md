# espn-get-nfl-schedule

Gets upcoming NFL games for a specified team with dates and opponents.

## Voice Triggers

- "Hey Cal, When do the Seahawks play next?"
- "Hey Cal, What's the Chiefs schedule?"

## Required Services



## Setup

No environment variables required.

No credentials required.

## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "espn-get-nfl-schedule"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s espn-get-nfl-schedule
```

## Usage

Get upcoming games for an NFL team. Required parameter: team (e.g., 'Seahawks', 'Patriots', 'SEA'). Examples: 'when do the Seahawks play next', 'Chiefs schedule', 'next Cowboys game'.
