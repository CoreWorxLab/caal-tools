# github-get-caal-repo-activity2

Gets open pull requests, issues, and unread notifications for a GitHub repository.

## Voice Triggers

- "Hey Cal, What's happening on the CAAL repo?"
- "Hey Cal, Check my GitHub activity"

## Required Services

GitHub

## Setup

No environment variables required.

### n8n Credentials

- **GITHUBAPI_CREDENTIAL** (`githubApi`)
  - n8n credential type: githubApi


## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "github-get-caal-repo-activity2"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s github-get-caal-repo-activity2
```

## Usage

Check CoreWorxLab/CAAL for actionable items: open PRs, open issues, and unread notifications.
